# RECODE IDE

Welcome to **RECODE IDE** – a highly customized, fast, and beautiful code editor fork based on VS Code OSS.

## Features
- **Kilo Code Integration**: Seamless AI pair programming built directly into the secondary sidebar.
- **Catppuccin Mocha Aesthetics**: A gorgeous dark theme integrated out of the box with custom translucent UI elements.
- **Optimized Core**: Telemetry disabled, clean UI, and fast startup times.

## Installation
1. Go to the **Releases** section of this repository and download the `RECODE IDE <version>.dmg` file.
2. Open the `.dmg` file and drag `RECODE IDE.app` into your Applications folder.
3. Open the IDE once to initialize the data folders, then close it.
4. Copy the `settings.json` file from this repository to your user settings directory:
   - **macOS**: `~/Library/Application Support/RECODE IDE/User/settings.json`
5. Re-open RECODE IDE and enjoy!

## Optional: Enable macOS Glass Effect (Vibrancy)
By default, the IDE features internal panel transparency. If you want the true macOS "frosted glass" effect where your desktop wallpaper blurs behind the editor window, we highly recommend installing the [Vibrancy Continued](https://open-vsx.org/extension/syler/vibrancy-continued) extension.

**How to enable:**
1. Open the Extensions tab (`Cmd+Shift+X`) in RECODE IDE.
2. Search for `Vibrancy Continued` and install it.
3. Reload the window when prompted.
4. Enjoy the true macOS blur!
